### Get `started`

```console
	git clone https://github.com/luamoris/react-faq-card-info
	cd react-faq-card-info
	npm i
	npm start
```

### For `build`

```console
	npm run build
```

### For `test`

```console
	npm test
```

### For `eject`

**Note: this is a one-way operation. Once you `eject`, you can’t go back!**

```console
	npm run eject
```

### For `lint`

- [stylelint][stylelint] - lint SCSS files

```console
	npm run stylelint
```
